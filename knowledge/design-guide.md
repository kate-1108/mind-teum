# Mind.teum 디자인 가이드

> 인스타그램 카드뉴스 자동생성을 위한 시각 디자인 기준서예요.
> 레퍼런스: @gamsung_train (감성기차), @powerofpositivity

---

## 1. 카드 규격

| 항목 | 값 |
|------|-----|
| 사이즈 | 1080 × 1350px (4:5 비율, 캐러셀 표준) |
| border-radius | 14px |
| border | 0.5px solid (버전별 서피스 컬러) |
| 패딩 (상하) | 80px |
| 패딩 (좌우) | 72px |
| 최대 슬라이드 수 | 10장 |

---

## 2. 레이아웃 구조

```
┌─────────────────────────────┐
│                             │  ← 상단 여백 80px
│   — 오늘의 마음 —           │  ← 태그 (상단에서 80px)
│                             │
│                             │
│                             │
│   본문 텍스트               │  ← 수직 중앙 정렬
│   두 번째 줄                │
│   세 번째 줄                │
│                             │
│                             │
│                             │
│   — @mind.teum —            │  ← 핸들 (하단에서 60px)
│                             │  ← 하단 여백 80px
└─────────────────────────────┘
```

### 레이아웃 원칙
- 본문 텍스트는 수직 중앙 정렬
- 태그는 상단 고정, 핸들은 하단 고정
- 텍스트 중앙 정렬 (text-align: center)
- 여백을 넉넉하게 — 텍스트가 카드의 60% 이하를 차지하도록

## 3. 배경 시스템

### 3-1. 단색 배경 (기본)
컬러 시스템 5가지 버전 그대로 적용 → `knowledge/color-system.md` 참조

### 3-2. 종이 질감 효과 (공통 적용)
```css
background-image: url("data:image/svg+xml,...");
/* fractalNoise 필터로 미세한 종이 질감 표현 */
/* opacity: 0.035 (라이트) / 0.05 (다크) */
```

### 3-3. 풍경 사진 오버레이 (선택적)
- 사진 위에 반투명 컬러 오버레이 적용
- 라이트: `rgba(253, 252, 250, 0.75)` 오버레이
- 다크: `rgba(14, 22, 36, 0.72)` 오버레이
- 텍스트 가독성 확보 필수

```css
.card-photo {
  background-image: linear-gradient(
    rgba(14, 22, 36, 0.72),
    rgba(14, 22, 36, 0.72)
  ), url('../backgrounds/dark/night-01.jpg');
  background-size: cover;
  background-position: center;
}
```

### 3-4. CSS 그라디언트 배경 (사진 대체)
```css
/* 노을 느낌 */
background: linear-gradient(160deg, #f5c5a3 0%, #e8a0b0 50%, #c97eb2 100%);

/* 새벽 느낌 */
background: linear-gradient(160deg, #0e1624 0%, #1a2a45 50%, #2a3a5a 100%);

/* 맑은 하늘 */
background: linear-gradient(160deg, #e8f0fa 0%, #d0e4f5 50%, #b8d4f0 100%);

/* 숲 새벽 */
background: linear-gradient(160deg, #101c18 0%, #1a2e26 50%, #243a30 100%);
```

## 5. 텍스트 스타일링

### 5-1. 기본 구성
```
— 오늘의 마음 —     ← 태그 (포인트 컬러)

본문 첫 번째 줄      ← 본문 (메인 텍스트 컬러)
본문 두 번째 줄
본문 세 번째 줄

— @mind.teum —      ← 핸들 (캡션 컬러)
```

### 5-2. 강조 표현 옵션
- **형광 하이라이트** (powerofpositivity 스타일)
```css
.highlight {
  background: linear-gradient(transparent 60%, #f5e642 60%);
  padding: 0 4px;
}
```
- **컬러 포인트 단어** (감성기차 스타일)
```css
.point-word { color: var(--mt-point); }
```

### 5-3. 줄바꿈 원칙
- 한 줄에 최대 13자 (의미 단위로 줄바꿈)
- 3~5줄 구성 권장
- 마침표 최소화, 쉼표·줄바꿈으로 여운

## 6. 슬라이드 구성 패턴

### 패턴 A — 감성 단문형 (감성기차 스타일)
```
슬라이드 1: 훅 문장 (가장 강한 한 줄)
슬라이드 2~4: 감성 전개
슬라이드 5: 위로·마무리 문장
슬라이드 6: CTA ("저장해두고 꺼내봐요 🤍")
```

### 패턴 B — 긍정 선언형 (powerofpositivity 스타일)
```
슬라이드 1: 핵심 메시지 1장
(단독 카드, 1카드 1메시지)
```

### 패턴 C — 정보+감성 혼합형
```
슬라이드 1: 주제 소개
슬라이드 2~5: 내용 전개 (번호 없이 자연스럽게)
슬라이드 6~7: 실천 제안
슬라이드 8: 마무리 감성 문장
```

## 9. 레퍼런스 계정 디자인 분석

### @gamsung_train (감성기차)
- **배경**: 종이 질감 흰 배경 또는 다크 배경, 실사 사진 오버레이 혼용
- **텍스트**: 손글씨 폰트, 중앙 정렬, 넉넉한 여백
- **구성**: 캐릭터 일러스트 + 감성 문구 조합
- **색상**: 단색 위주, 포인트 컬러로 특정 단어 강조
- **어체**: 해요체·해체 혼용, 줄바꿈으로 호흡감

### @powerofpositivity
- **배경**: 풍경 사진(노을·꽃밭·바다·하늘) 오버레이가 주력
- **텍스트**: 필기체 + 볼드 산세리프 믹스
- **구성**: 1카드 1메시지 원칙 철저히 준수
- **강조**: 노란 형광 하이라이트로 핵심 문장 강조
- **어체**: 독자에게 직접 말 거는 2인칭 문장

## 10. 금지 사항

- ❌ 카드 한 장에 텍스트 과다 (6줄 초과)
- ❌ 폰트 3가지 이상 혼용
- ❌ 배경과 텍스트 명도 대비 부족 (가독성 저해)
- ❌ 이모지 사용 (텍스트만으로 감성 전달)
- ❌ 로고/워터마크 상단 배치 (하단 핸들로 통일)
