# Mind.teum 컬러 시스템

> 라이트 3가지 + 다크 2가지, 총 5가지 버전을 주제에 따라 선택해서 사용해요.

---

## 버전 A — Pure Cream (라이트)

> 거의 흰색이지만 차갑지 않은 크림 베이스. 미드나잇 블루 포인트로 차분하고 깊은 느낌.

| 항목 | HEX | 용도 |
|------|-----|------|
| 베이스 배경 | `#fdfcfa` | 카드 기본 배경 |
| 서피스 배경 | `#f7f5f1` | 보조 카드 배경 |
| 포인트 | `#1c2d4a` | 태그, 강조 텍스트 (미드나잇 블루) |
| 본문 텍스트 | `#221e1a` | 메인 텍스트 (웜 차콜) |
| 캡션 / 핸들 | `#786e62` | @mind.teum 등 보조 텍스트 (웜 그레이) |

**적합한 주제:** 긍정 · 성장 · 동기 · 일상 공감 · 자기애

```css
--mt-base: #fdfcfa;
--mt-surface: #f7f5f1;
--mt-point: #1c2d4a;
--mt-text: #221e1a;
--mt-caption: #786e62;
```

## 버전 B — Cloud White (라이트)

> 하늘빛이 살짝 감도는 밝고 산뜻한 화이트. 포인트와 캡션이 같은 블루 계열로 통일감 있음.

| 항목 | HEX | 용도 |
|------|-----|------|
| 베이스 배경 | `#fafbfc` | 카드 기본 배경 |
| 서피스 배경 | `#f0f3f6` | 보조 카드 배경 |
| 포인트 | `#1c2d4a` | 태그, 강조 텍스트 (미드나잇 블루) |
| 본문 텍스트 | `#1e2028` | 메인 텍스트 (쿨 차콜) |
| 캡션 / 핸들 | `#475a6e` | @mind.teum 등 보조 텍스트 (블루 그레이) |

**적합한 주제:** 긍정 에너지 · 맑은 감성 · powerofpositivity 스타일

```css
--mt-base: #fafbfc;
--mt-surface: #f0f3f6;
--mt-point: #1c2d4a;
--mt-text: #1e2028;
--mt-caption: #475a6e;
```

## 공통 디자인 요소

| 항목 | 값 |
|------|-----|
| 폰트 (본문) | 강원교육새음체 or 국립공원 꼬미 (주제별 선택) |
| 폰트 (태그·핸들) | SUIT Light / letter-spacing 0.08~0.12em |
| 줄간격 | 2.0 |
| 카드 상단 태그 | `— 오늘의 마음 —` 또는 `— 마인드틈 —` |
| 카드 하단 핸들 | `— @mind.teum —` |
| 배경 효과 | CSS fractalNoise 노이즈 필터 (종이 질감) |
| 카드 border-radius | 14px |
| 카드 border | 0.5px solid (각 버전 서피스 컬러 기준) |
